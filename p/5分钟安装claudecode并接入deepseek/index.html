概述

Claude Code 是 Anthropic 推出的命令行 AI 编程助手，内置多种 Skills 插件，通过不同组合可以完成许多 AI 单独做不到的事。接入国产大模型 DeepSeek V4 后，可以大幅降低使用成本。

核心用途：

  • 构建个人知识库，自动化整理学习笔记
  • 按时搜集行业资讯并整理成文稿
  • 自动发布到各大社交媒体平台

第一步：安装运行环境

1.1 安装 Node.js

前往 Node.js 官网 下载 Windows 安装包（.msi），双击后一路下一步。

验证安装：

1
2

node -v
npm -v

1.2 配置 npm 国内镜像 

1

npm config set registry https://registry.npmmirror.com/
💡 为什么要配置镜像？

默认 npm 源在国外，下载速度很慢。配置国内镜像后安装速度可提升 10 倍以上。

1.3 安装 Git

前往 Git 官网 下载对应版本安装包，一路下一步。

验证安装：

1

git -v

1.4 安装 CC-Switch

CC-Switch 用于将第三方模型接入 Claude Code。

前往 GitHub Release 页面 下载对应版本（如 Windows MSI），双击安装，一路下一步。

第二步：安装 Claude Code 

1

npm install -g @anthropic-ai/claude-code

验证：

1

claude --version

跳过登录（使用第三方 API）

  1. 启动一次 Claude Code 后关闭
  2. 在用户目录下找到 .claude.json 文件
  3. 添加以下字段（注意上一字段尾部需加英文逗号）：
1

"hasCompletedOnboarding": true
  1. 重新执行 claude 启动，选择信任当前文件夹即可
⚠️ 常见错误

如果忘记在上一字段末尾加逗号，会导致 JSON 解析错误，Claude Code 无法启动。

第三步：接入 DeepSeek

3.1 获取 API Key

  1. 打开 DeepSeek 开放平台，注册登录
  2. 确保账户有余额
  3. 右侧 API Keys → 创建 API Key → 复制保存（关闭后无法再次查看）
🚨 密钥安全

密钥不要分享给他人，否则会消耗你的余额。

3.2 配置 CC-Switch

  1. 打开 CC-Switch，点击右上角 + 号
  2. 模型预设选择 DeepSeek
  3. 填入 API Key
  4. 模型名称统一改为：deepseek-v4-pro[1m]（上下文 1M，Claude Code 默认使用 128K）
  5. 点击添加

3.3 验证模型

在 Claude Code 对话框中输入：

1

/model

可查看当前可用模型，或直接问"你当前是什么模型"确认接入成功。

第四步：实战示例

  1. 桌面新建一个文件夹
  2. 在地址栏输入 CMD 回车
  3. 执行 claude 启动，选择信任当前文件夹
  4. 输入需求：使用 HTML + JS + CSS 做一个 Todo 软件
  5. 运行中会申请工具权限，选择 Yes 即可
  6. 完成后让它帮你打开页面预览
💡 小技巧

可以用中文直接描述需求，Claude Code 支持中英文混合输入。

常见问题

执行 claude 后提示找不到命令？

检查 Node.js 是否安装成功（node -v），以及 npm 全局路径是否在系统 PATH 中。

启动后要求登录 Anthropic 账号？

说明 .claude.json 中的 hasCompletedOnboarding 未正确配置，请回到第二步检查。

模型调用报错 / 无响应？

检查 DeepSeek 账户余额是否充足，以及 CC-Switch 中 API Key 是否正确填写。
