C 碟爆滿？用其他磁碟為 C 碟擴容

2026年5月24日 星期日 閱讀時間: 3 分鐘

C 碟飄紅是 Windows 用戶的"老毛病"了——系統更新緩存、臨時檔案、桌面檔案、微信聊天記錄……不知不覺就把 C 碟塞得滿滿當當。換硬碟成本高，清理治標不治本，最一勞永逸的辦法就是：從其他磁碟借空間給 C 碟。

📦 準備工作

⚡ 一句話概括：把 Windows 裝進一個"休眠箱"裡操作，這樣 C 碟才不會被佔用。

① 下載微 PE 工具箱

微 PE 工具箱——輕量級 PE（預安裝環境）工具，乾淨無捆綁。

② 安裝到系統

打開下載好的程式，點擊 「立即安裝進系統」，它會自動寫入引導項。

③ 重啟進入引導頁面

安裝完成後重啟電腦，啟動時會看到系統引導選擇頁面。

⏱ 注意：倒數只有幾秒鐘，超時後會自動進入當前系統，盯緊螢幕快速選擇！

④ 進入 PE 環境

方向鍵選擇 微 PE 工具箱，回車進入。

💡 為什麼要進 PE？

進入 PE 環境後，你的 Windows 系統處於完全"休眠"狀態——C 碟沒有被任何進程佔用，此時才能安全地調整分區大小。這就像一個手術，病人必須麻醉後才能動刀。其他分區工具也是同樣的原理。

🛠️ 開始擴容

進入 PE 後，桌面會自動彈出工具選單，點擊 「分區助手」。

第一步 · 查看磁碟佈局

分區助手打開後，你會看到所有磁碟分區的直觀佈局。確認一下哪個碟有空餘空間可以"支援"C 碟。

第二步 · 選擇要縮小的碟

這裡以 Software（D 碟） 為例，我們要從 D 碟"切"出一部分空間給 C 碟。

💡 選哪個碟好？

  • ✅ 優先選 空間充裕、且不常用的大碟
  • ❌ 盡量別動系統恢復分區或 EFI 分區

第三步 · 調整分區

右鍵點擊 D 碟 → 「調整分區」。

第四步 · 縮小碟符

在彈出的視窗中，拖動滑塊或直接輸入數值，決定從 D 碟分出多少空間。

✅ 範例：你想給 C 碟擴充 50GB，就在這裡縮小 50GB

⚠️ 注意：縮小的數值不要超過 D 碟當前已用空間，否則會報錯。

確認無誤後點擊 「確定」。

第五步 · 合併給 C 碟

此時你會看到 D 碟旁邊多了一塊 「未分配空間」。

步驟 操作
1 右鍵點擊 C 碟 → 「調整分區」
2 將滑塊拖到 最右側，把全部未分配空間併入 C 碟
3 點擊 「確定」


第六步 · 提交任務

點擊左上角的 「提交」。

⚠️ 關鍵提醒

前面所有的調整都只是**“計劃”**，並沒有實際生效。

只有點擊「提交」後，系統才會真正執行分區操作。確認無誤再提交！

第七步 · 等待執行

分區助手會開始執行調整，這個過程可能需要幾分鐘到十幾分鐘，具體取決於你的磁碟大小和速度。

⚠️ 🔴 千萬不要中途斷電或強制關機！
否則可能導致分區表損壞，資料遺失
建議插上電源適配器（筆記本用戶）
去泡杯茶，耐心等它跑完 🍵

第八步 · 完成 🎉

看到成功提示後，擴容就完成了！

🚀 重啟與收尾

順序 操作
① 點擊 「重啟」
② 在引導介面選擇你的 Windows 系統 正常進入
③ 打開 此電腦，確認 C 碟空間已經變大 ✅


🧹 最後一步：如果你不想每次開機都看到系統選擇介面，打開 微 PE 工具箱 → 「卸載」 即可恢復如初。

📌 小提示

場景 建議
🟢 C 碟空間太小（< 30GB） 建議至少擴充到 80 ~ 100GB
🟡 D 碟也沒有多餘空間 清理 D 碟，或外接硬碟轉移大檔案
🔴 操作前擔心資料遺失 強烈建議先備份重要檔案！ 雖然分區助手很穩定，但小心駛得萬年船

✨ 以上就是完整的 C 碟擴容教程，希望你的電腦從此遠離"飄紅"！
